# 🤖 Black-Wall AI

Black-Wall AI is a simple ChatGPT-inspired chatbot that runs with a local language model using LM Studio. I built this project to learn how AI models can be integrated into a web application while improving my frontend and JavaScript skills.

This is one of my first AI projects, and it has been a great way to understand concepts like DOM manipulation, APIs, asynchronous JavaScript, and debugging(lots).

## Features

- ChatGPT-inspired interface
- Local AI integration using LM Studio
- User and AI chat messages
- Auto-scrolling chat window
- Prevents empty messages
- Simple and clean dark theme

## Built With

- HTML
- CSS
- JavaScript
- LM Studio
- GPT-OSS-20B
- Git & GitHub

## Running the Project

1. Clone the repository.
2. Open the project in VS Code.
3. Start a model in LM Studio.
4. Enable the Local Server and turn on CORS.
5. Run the project using Live Server.
